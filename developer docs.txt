///////////////////
// About the app

- full stack web app where users can upload images
- follow friends
- set account type to public / private
- like others images 
- comment on others posts 
- receive notification 
- view others profile 

///////////////////
// Tech Used 

frontend: html,css,js 
backend: laravel 
database: mysql

//////////////////
// Roles and functions

Roles: 
- User 

Features: 
- User can create account using username and password
- Profile page 
-- uploaded photos
-- profile details
-- edit profile 
--- profile picture 
--- email 
--- account type "public" , "private"
- Settings page 
-- user can switch light/dark theme 
- Upload image page
-- user can upload images from this page with caption
- Inbox page 
-- user can see received messages in this page 
- Social page
-- user can search other users from this page 
--- see images of public account 
--- follow / unfollow users. 
- Feed page 
-- User can see public posts in this page (click to view)
-- user can like , comment
